# 項目組織

隨著項目規模的增長，良好的代碼組織變得至關重要。本章節將介紹：

- **7-1 模式** - 業界廣泛採用的 SCSS 項目架構模式
- **ITCSS 方法論** - 倒三角 CSS 架構方法
- **命名規範** - BEM、OOCSS 等命名方法論

學會這些組織方法，將幫助你建立可維護、可擴展的 SCSS 項目結構。
